# Discord Bot

## How to use?
* Read and modify the /data/dummy_xxx.ron files and make the needed modifications
    * [Direct link](/data)
* ( ```cargo build --release``` )
* ```cargo run --release```

## Requirements
* [Rust](https://www.rust-lang.org/) (latest <u class="red">**stable**</u> version)

## Note
Do not build using nightly rustc, it will not work
